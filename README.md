# SOAL2_PersiapanUKL
Program Faktorial menghitung hasil perkalian berurutan dari 1 hingga bilangan input.  
Program VolumeTabung menggunakan fungsi (method) dengan parameter r dan t untuk menghitung volume tabung berdasarkan rumus π × r² × t.
